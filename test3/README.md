# Test 3 â€” CSV Processing: Filter Properties Below Average Price per Sqft

This script reads a CSV file containing real estate sale prices and generates a
new CSV file that includes only those properties where the **price per square foot**
is **less than the average price per square foot**.

---

#  Files

| File | Description |
|------|-------------|
| `process_sales.py` | Python script to filter the dataset |
| `sales-data.csv` | Input CSV file (must be uploaded) |
| `filtered_sales.csv` | Output generated by the script |

---

#  How to Run

Place your CSV file inside the `test3` folder with the name:

```
sales-data.csv
```

Then run:

```bash
python3 process_sales.py
```

Output:

```
filtered_sales.csv
```

---

#  What the Script Does

1. Reads input CSV  
2. Calculates `price_per_sqft` = SalePrice / Sqft  
3. Computes the dataset average  
4. Filters all records where `price_per_sqft < average`  
5. Saves them to a new CSV  

---

#  Requirements

- Python 3
- `pandas` library (`pip install pandas`)
